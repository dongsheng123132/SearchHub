# SearchHub 隐私政策

**最后更新：2025年11月27日**

## 概述

SearchHub 是一款单一用途的多引擎搜索工具。它不会收集、存储、出售或传输个人数据。

## 数据处理

- **无需注册账号**
- **无需追踪**
- **无需分析**

设置存储在本地或通过 `chrome.storage.sync` 存储（主题、选定的引擎、组合、上次查询）。

所选文本仅用于执行用户发起的搜索，不会持久保存或发送到任何服务器。

## 权限说明

| 权限 | 用途 |
|------|------|
| **tabs**、**tabGroups** | 打开和组织结果标签 |
| **sidePanel** | 显示垂直标签概览 |
| **contextMenus**、**activeTab**、**scripting** | 根据用户手势运行搜索并读取 Alt+Q 的当前选择 |
| **notifications** | 显示本地指导信息 |
| **system.display** | 可选的窗口布局，用于并排显示结果 |
| **host_permissions** (`<all_urls>`) | 打开跨引擎/域的结果页面；不收集页面内容 |

## 安全

- **没有远程代码**：所有脚本都打包在扩展程序包中
- **禁止外部数据传输**
