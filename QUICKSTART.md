# SearchHub 快速开始指南

## 安装步骤

### 1. 准备图标文件

由于浏览器扩展需要图标文件，请先准备图标：

**选项A：从 SeekAll 复制图标**
```bash
# 将 SeekAll 的图标复制到 SearchHub
cp "CMFGOMDHMKNHBGBDNAGKIJKDAIFNECMA_1_6_0_0 (1)/assets/icons/"*.png SearchHub/icons/

# 重命名文件
cd SearchHub/icons
mv icon16.png icon16.png  # 如果名称不同，请调整
mv icon32.png icon32.png
mv icon48.png icon48.png
mv icon128.png icon128.png
```

**选项B：使用在线工具生成**
1. 访问 https://favicon.io/
2. 上传一张图片或使用文字生成
3. 下载生成的图标包
4. 解压并复制 icon16.png, icon32.png, icon48.png, icon128.png 到 SearchHub/icons/ 文件夹

### 2. 加载扩展到浏览器

1. 打开 Chrome 或 Edge 浏览器
2. 在地址栏输入：`chrome://extensions/` 或 `edge://extensions/`
3. 打开右上角的"开发者模式"开关
4. 点击"加载已解压的扩展程序"按钮
5. 选择 `SearchHub` 文件夹
6. 确认加载成功

### 3. 固定扩展图标

1. 点击浏览器工具栏的拼图图标
2. 找到 SearchHub
3. 点击图钉图标固定到工具栏

## 基础功能测试

### 测试1：主页面

1. 点击工具栏的 SearchHub 图标（或按 Alt+S）
2. 应该看到美观的主页面，包含：
   - 搜索框
   - 快捷组合按钮
   - 分类的搜索引擎卡片
   - 亮色/暗色模式切换按钮

### 测试2：主题切换

1. 点击右上角的太阳/月亮图标
2. 观察页面从亮色切换到暗色（或相反）
3. 所有颜色应该平滑过渡

### 测试3：搜索引擎选择

1. 点击几个搜索引擎卡片
2. 被选中的卡片应该变成紫色背景
3. 底部显示已选择的引擎数量
4. 搜索按钮应该启用

### 测试4：执行搜索

1. 在搜索框输入关键词（如："AI 发展"）
2. 选择2-3个搜索引擎（如：Google, Bing, Perplexity）
3. 点击搜索按钮
4. 应该看到：
   - 打开多个新标签页
   - 标签页自动分组
   - 组标题显示搜索关键词
   - 每个标签页显示对应搜索引擎的结果

### 测试5：快捷组合

1. 点击"🤖 All AI"组合按钮
2. 所有 AI 搜索引擎应该被自动选中
3. 输入搜索关键词
4. 点击搜索
5. 应该在所有 AI 引擎中搜索

### 测试6：右键菜单搜索

1. 在任意网页上选中一段文字
2. 右键点击
3. 选择"Search with SearchHub"
4. 应该自动使用之前选择的引擎搜索该文字

### 测试7：快捷键

1. 选中网页上的文字
2. 按 Alt+Q (Windows) 或 Command+Shift+Q (Mac)
3. 应该自动搜索选中的文字

### 测试8：设置页面

1. 在主页点击右上角的齿轮图标
2. 应该打开设置页面
3. 测试以下功能：
   - 主题选择下拉框
   - 打开模式选择（新标签页/新窗口）
   - 各种开关按钮
   - 添加自定义组合
   - 添加自定义引擎

### 测试9：自定义搜索引擎

1. 在设置页面点击"Add Engine"
2. 填写：
   - 名称：知乎直达
   - URL：`https://www.zhihu.com/search?q=%s`
   - 描述：知乎搜索
3. 保存
4. 返回主页，应该看到新添加的引擎

### 测试10：导出/导入配置

1. 在设置页面点击"Export Settings"
2. 应该下载一个 JSON 文件
3. 修改一些设置
4. 点击"Import Settings"选择刚才下载的文件
5. 设置应该恢复到之前的状态

### 测试11：侧边栏

1. 执行一次搜索（会创建标签组）
2. 点击浏览器地址栏右侧的侧边栏图标
3. 或者在设置中开启"显示侧边栏"选项
4. 应该看到：
   - 当前窗口的所有标签组
   - 可以展开/折叠组
   - 点击标签可切换
   - 点击关闭按钮可关闭标签

## 常见问题

### Q1：扩展加载失败
**A：** 确保：
1. 图标文件存在于 icons 文件夹
2. manifest.json 没有语法错误
3. 所有 HTML/JS/CSS 文件都存在

### Q2：搜索没有反应
**A：**
1. 打开浏览器开发者工具（F12）
2. 查看 Console 是否有错误
3. 确保至少选择了一个搜索引擎
4. 检查搜索框是否有内容

### Q3：右键菜单没有出现
**A：**
1. 重新加载扩展
2. 刷新当前网页
3. 检查是否选中了文字

### Q4：快捷键不工作
**A：**
1. 检查快捷键是否与其他扩展冲突
2. 在 chrome://extensions/shortcuts 中查看和修改快捷键

### Q5：暗黑模式没有生效
**A：**
1. 检查设置中的主题选项
2. 如果选择"Auto"，会跟随系统主题
3. 手动切换为"Dark"

## 高级功能

### 自定义搜索 URL

搜索 URL 中使用 `%s` 作为搜索查询的占位符。例如：
- Google: `https://www.google.com/search?q=%s`
- GitHub: `https://github.com/search?q=%s`
- YouTube: `https://www.youtube.com/results?search_query=%s`

### 多窗口模式

1. 在设置中选择"打开模式"为"New Windows"
2. 搜索时会将不同引擎的结果并排显示在多个窗口中
3. 适合宽屏显示器使用

### 标签组管理

1. 搜索后自动创建标签组
2. 可以手动折叠/展开组
3. 可以在侧边栏中管理所有组
4. 关闭组内最后一个标签时，组会自动删除
