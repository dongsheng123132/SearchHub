# 垂直标签页功能指南

## 概述

SearchHub 的垂直标签页功能让您可以在侧边栏中管理所有打开的标签页，类似于 SeekAll 的体验，但功能更强大。

## 主要特性

### ✅ 自动打开侧边栏

搜索后，侧边栏会**自动打开**，显示所有标签页的垂直列表，无需手动操作。

### ✅ 完整的标签页视图

侧边栏显示：
- **分组的标签页** - 搜索创建的标签组，带有颜色标识和标题
- **未分组的标签页** - 其他所有打开的标签页
- **当前激活的标签** - 高亮显示正在查看的标签

### ✅ 强大的管理功能

1. **查看所有标签**
   - 显示标签的图标、标题和域名
   - 一目了然地看到所有打开的页面

2. **快速切换标签**
   - 点击任意标签即可切换到该页面
   - 当前激活的标签会高亮显示

3. **展开/折叠分组**
   - 点击分组标题可展开或折叠
   - 点击顶部的折叠按钮可一键折叠/展开所有分组

4. **关闭标签**
   - 鼠标悬停在标签上时显示关闭按钮
   - 点击即可快速关闭不需要的标签

5. **解散分组**
   - 鼠标悬停在分组标题上时显示解散按钮
   - 点击可将分组的标签变为普通标签

6. **实时更新**
   - 标签页的任何变化都会实时反映在侧边栏中
   - 包括标题更新、图标加载、新标签打开等

## 使用方法

### 基础操作

1. **打开侧边栏**
   - 执行搜索后自动打开
   - 或点击浏览器地址栏右侧的侧边栏图标

2. **查看标签列表**
   - 顶部显示最近的搜索关键词（如果有）
   - 下方按分组显示所有标签
   - 未分组的标签显示在底部的"Other Tabs"区域

3. **切换标签**
   - 直接点击标签项
   - 当前激活的标签会有紫色边框标识

4. **管理分组**
   - 点击分组标题展开/折叠
   - 悬停显示解散按钮，点击可解散分组
   - 使用顶部按钮一键展开/折叠所有分组

5. **关闭标签**
   - 悬停在标签上
   - 点击右侧的 × 按钮

### 高级技巧

1. **多窗口管理**
   - 侧边栏只显示当前窗口的标签
   - 每个窗口都有独立的侧边栏视图

2. **快速定位**
   - 标签显示域名，方便识别网站
   - 长标题会自动截断，鼠标悬停可查看完整标题

3. **搜索历史**
   - 侧边栏顶部显示最后一次搜索的关键词
   - 方便回顾搜索内容

4. **刷新按钮**
   - 如果列表未及时更新，可点击刷新按钮手动刷新

## 配置选项

在设置页面可以调整以下选项：

1. **Show Sidebar** (默认: 开启)
   - 搜索后是否自动显示侧边栏
   - 即使关闭，也可以手动打开侧边栏

2. **Group Tabs** (默认: 开启)
   - 是否自动将搜索结果分组
   - 关闭后标签不会自动分组

3. **Collapse Groups** (默认: 关闭)
   - 新创建的分组是否默认折叠
   - 开启后分组默认收起，需要点击展开

## 常见问题

### Q: 侧边栏没有自动打开怎么办？

**A:**
1. 检查设置中的"Show Sidebar"选项是否开启
2. 手动点击浏览器地址栏右侧的侧边栏图标
3. 重新加载扩展

### Q: 为什么有些标签没有显示图标？

**A:**
- 网站尚未加载完成
- 网站没有提供 favicon
- 会使用默认图标代替

### Q: 可以拖拽标签重新排序吗？

**A:**
- 当前版本暂不支持拖拽排序
- 可以通过 Chrome 原生的标签拖拽功能调整顺序
- 侧边栏会实时反映调整结果

### Q: 侧边栏更新不及时怎么办？

**A:**
- 点击右上角的刷新按钮
- 大多数情况下会自动实时更新

### Q: 可以在侧边栏搜索吗？

**A:**
- 当前版本暂不支持侧边栏内搜索
- 可以点击底部的"Open SearchHub"按钮进行新搜索

### Q: 侧边栏占用太多空间怎么办？

**A:**
- 可以调整浏览器侧边栏的宽度
- 点击侧边栏边缘拖拽即可调整
- 不需要时可以关闭侧边栏

## 最佳实践

1. **保持分组清晰**
   - 搜索不同主题时会创建不同颜色的分组
   - 及时关闭不需要的标签，保持列表简洁

2. **使用折叠功能**
   - 分组较多时，折叠暂时不需要的分组
   - 使用一键折叠/展开功能快速整理

3. **合理使用解散功能**
   - 搜索完成后，可以解散分组
   - 让标签回归正常管理

4. **配合快捷键**
   - Alt+S 打开 SearchHub 进行新搜索
   - Alt+Q 快速搜索选中的文字
   - Ctrl+Tab 在标签间快速切换

## 技术细节

### 实时监听

侧边栏会监听以下事件并实时更新：

- `tabs.onActivated` - 标签激活
- `tabs.onUpdated` - 标签更新（标题、图标等）
- `tabs.onCreated` - 新标签创建
- `tabs.onRemoved` - 标签关闭
- `tabs.onMoved` - 标签移动
- `tabs.onAttached` - 标签附加到窗口
- `tabs.onDetached` - 标签从窗口分离
- `tabGroups.onCreated` - 分组创建
- `tabGroups.onUpdated` - 分组更新
- `tabGroups.onRemoved` - 分组删除

### 性能优化

- 只监听当前窗口的标签变化
- 仅在显著变化时刷新（标题、图标、完成加载）
- 使用事件委托减少事件监听器数量
- 优化 DOM 操作，减少重绘
