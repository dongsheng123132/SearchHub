# Screenshot Guide for Chrome Web Store

## 📸 Required Screenshots: 3-5 images (1280x800 recommended)

### How to Take Perfect Screenshots

#### Step 1: Prepare Your Browser
1. Open SearchHub extension
2. Set browser window to 1280x800:
   - Press F12 to open DevTools
   - Press Ctrl+Shift+M to toggle device toolbar
   - Select "Responsive" and set to 1280x800
   - OR use Console: `window.resizeTo(1280, 800)`

#### Step 2: Clean Up Your Interface
- Close unnecessary tabs
- Use light theme for better visibility (toggle in extension)
- Make sure all text is in English
- Fill search box with example: "artificial intelligence"

## 🎨 Screenshot Best Practices

### Quality
- ✅ 1280x800 or 640x400 resolution
- ✅ PNG format (better quality than JPG)
- ✅ Clear, crisp text (no blur)
- ✅ Good contrast and lighting

### Content
- ✅ Show real, useful features
- ✅ Use realistic example searches
- ✅ Clean, professional appearance
- ✅ English language UI
- ❌ No personal information
- ❌ No browser bookmarks/extensions in view
- ❌ No system notifications

### Composition
- ✅ Center the main content
- ✅ Show complete UI elements (no cut-off)
- ✅ Logical progression (screenshot 1→2→3)
- ✅ Consistent theme across all screenshots

## 🛠️ Tools to Take Screenshots

### Windows
- **Snipping Tool**: Win+Shift+S
- **Snip & Sketch**: Windows + Shift + S

### Mac
- **Full Screen**: Cmd+Shift+3
- **Selection**: Cmd+Shift+4
- **Window**: Cmd+Shift+4, then Space

- **Awesome Screenshot**
- **Nimbus Screenshot**

## 🎯 Where to Upload

After taking screenshots:
1. Save in a new folder: `SearchHub/store-assets/screenshots/`
2. Name them: `screenshot-1.png`, `screenshot-2.png`, etc.
3. Upload to Chrome Developer Dashboard during submission

**Maximum file size:** 2MB per screenshot
**Format:** PNG or JPEG (PNG recommended)
**Dimensions:** 1280x800 or 640x400
