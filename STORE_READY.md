## 📋 What You Need to Do (30 mins)

### 1. Update Two Fields in manifest.json (2 mins)
Open `manifest.json` and change:
```json
"author": "Your Name",  // ← Change to your name
"homepage_url": "https://github.com/yourusername/searchhub",  // ← Your URL
```

### 2. Take 3-5 Screenshots (15 mins)
Follow `SCREENSHOT_GUIDE.md` for details. Quick version:

**Required Screenshots:**
1. Main interface - Show search box + engine cards
2. Search results - Show multiple tabs opened
3. Combo feature - Show combo buttons + selected engines

**Optional Screenshots:**
4. Edit mode - Show edit/delete buttons
5. Add engine dialog - Show the modal form

**How:**
- Open extension
- Press Win+Shift+S (Windows) or Cmd+Shift+4 (Mac)
- Save as PNG (1280x800 recommended)

### 3. Host Privacy Policy (5 mins)
**Easiest way: GitHub Gist**
1. Go to https://gist.github.com/
2. Sign in with GitHub
3. Create new gist
4. Paste content from `PRIVACY_POLICY.md`
5. Make it public
6. Copy the URL

**Save this URL - you'll need it for submission!**

### 4. Create ZIP Package (5 mins)
**Include these folders/files:**
```
SearchHub.zip
├── manifest.json
├── index.html
├── sidebar.html
├── settings.html
├── background.js
├── content.js
├── content.css
├── popup.js
├── icons/
├── scripts/
├── styles/
└── _locales/
```

**EXCLUDE:**
- All .md files
- store-assets/
- Any test files

**How to create:**
- Windows: Select all files → Right-click → Send to → Compressed folder
- Mac: Select all files → Right-click → Compress

## 🎯 Step-by-Step Submission

### Quick Version:

1. **Register** ($5 fee)
   - Go to https://chrome.google.com/webstore/devconsole
   - Pay $5 USD registration

2. **Upload ZIP**
   - Click "New Item"
   - Upload your ZIP file

3. **Fill Details**
   - Paste extension name
   - Paste descriptions
   - Upload 3-5 screenshots
   - Add privacy policy URL

4. **Submit**
   - Click "Submit for Review"
   - Wait 1-7 days

5. **Done!** 🎉
